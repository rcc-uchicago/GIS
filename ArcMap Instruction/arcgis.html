Leave default Python directory C:\Python27


ArcGIS Pro uses Python 3.0+, while ArcMap uses Python 2.7. Python 3. and Python 2. are different languages and many functions are not compatible with each other. For ArcMap users, please check out here (list of ArcMap tools not available in ArcGIS pro:  https://pro.arcgis.com/en/pro-app/latest/tool-reference/appendices/unavailable-tools.htm ) to find the functions that are not supported in ArcGIS pro.


Learn more about the differences between Python 3 and Python 2: https://www.guru99.com/python-2-vs-python-3.html

After the installation is complete, the ArcGIS Administrator Wizard is launched. Select your product type as Advanced (ArcInfo) and click Authorize Now..


The ArcGIS Administrator Wizard is only launched the first time you install ArcGIS Desktop. After you have completed the ArcGIS Administrator Wizard, you can access the ArcGIS Administrator by clicking Start > Programs > ArcGIS > ArcGIS Administrator.

10.

Highlight the radio button next to “Authorize with ESRI now using the Internet” and click NEXT.

The next few dialog boxes will ask for your personal information, or the fields may already be filled in. Click NEXT after completing each dialog. DO NOT manipulate the authorization numbers or the number of licenses, or the software will not function.

Click FINISH when completed.

11.

ArcMap Setup

  • Go to Start>Programs>ArcGIS>ArcMap. ArcMap will open.
  • Click OK.
  • In the top toolbar, go to Customize>Extensions.
  • Check all boxes.
  • Click CLOSE.
  • Close ArcMap.

ArcCatalog Setup

  • Go to Start>Programs>ArcGIS>ArcCatalog. ArcCatalog will open.
  • In the top toolbar, go to Customize>Extensions.
  • Check all boxes.
  • Click CLOSE.
  • Close ArcCatalog.
